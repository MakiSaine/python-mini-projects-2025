## About the Project
The goal of this project was to explore how programming logic and object-oriented thinking can be applied in small, simple programs that still have structure and real functionality.  
Each script focuses on a specific area of Python, such as file handling, data validation, loops, and user interaction.  
Together they show the process of learning how to write clear and reusable code while understanding the importance of testing and debugging.

---

## Projects Included

### Guess Game
A simple word guessing game that selects a random word from a text file and lets the user guess one letter at a time.  
It keeps track of correct and wrong guesses, shows progress after each attempt, and ends when the player finds the word or runs out of chances.  
This small project demonstrates how to work with files, random values, loops, and conditions.  

**Files:** `guess_game.py` and `words.txt`

---

### Library Management System
A program built to simulate a basic digital library.  
It allows users to add and remove books, check them in and out, and view the full collection.  
This project focuses on object-oriented programming using two main classes: `Book` and `Library`.  
It shows how classes, attributes, and methods can be used to manage data in a structured way.

**Files:** `library.py` and `library_script.py`

---

### Bank Account Simulator
This console program lets a user open an account, deposit and withdraw money, and calculate interest.  
It also shows the importance of input validation and exception handling, making it a good example of user interaction in a loop-based system.  
Each option is presented through a simple text-based menu.

**File:** `bank.py`

---

### Palindrome Checker
A short program that checks whether a word or phrase is a palindrome.  
It removes spaces and punctuation before comparison and tells the user if the text reads the same backward and forward.  
This project highlights simple string manipulation and logic flow.

**File:** `palindrome.py`

## How to Run
You can run any of the scripts directly in your terminal.  
Make sure Python is installed on your system.

```bash
python guess_game.py
python library_script.py
python bank.py
python palindrome.py
```

If you are running the guessing game, make sure the file `words.txt` is in the same folder.  
Each program runs in the terminal and requires no extra libraries or setup.
